# E-Commerce Frontend

Modern e-commerce frontend built with React, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Docker

```bash
docker build -t ecommerce-frontend .
docker run -p 80:80 ecommerce-frontend
```
